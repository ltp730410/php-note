##LINUX开发环境的搭建

####Nginx

- **nginx安装**

```
	sudo apt-get update
	sudo apt-get install nginx
```
- **nginx卸载**

	卸载方法一：
	
	```
	删除nginx，保留配置文件
	sudo apt-get remove nginx
	
	删除配置文件
	rm -rf /etc/nginx
	```
	
	卸载方法二：
	
	```
	删除nginx连带配置文件
	sudo apt-get purge nginx # Removes everything.

	卸载不再需要的nginx依赖程序
	sudo apt-get autoremove
	```
